# TextSwitcher 🔄

**A lightweight alternative to LangOver**

Fast and stable keyboard layout converter for Hebrew ↔ English. Born out of frustration with LangOver crashes and instability.

## ✨ Features

- 🔄 **F10** - Instantly convert Hebrew ↔ English layouts
- 🔤 **F6** - Swap UPPER ↔ lower case
- 🪶 **Lightweight** - Only ~150 KB
- ⚡ **Fast** - No lag, instant conversion
- 💪 **Stable** - No crashes, no memory leaks
- 🎯 **Simple** - Does one thing really well

## 🎬 How It Works

Ever typed akuo instead of שלום?

1. Select the text (Ctrl+A)
2. Press **F10**
3. ✨ Fixed!

## 📥 Installation

1. Download from [Releases](../../releases)
2. Install [.NET 8 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/8.0/runtime)
3. Run TextSwitcher.exe

## 🛠️ Build from Source

bash
git clone https://github.com/fusion-creator/TextSwitcher.git
cd TextSwitcher
dotnet build -c Release
